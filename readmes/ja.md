# <img src="https://github.com/lipis/flag-icons/raw/refs/heads/main/flags/4x3/jp.svg" width="30"> ZapUnlocked-API

プロフェッショナルな WhatsApp 自動化 API。シンプル、高速、そして強力です。

**主な機能：**
- ✅ 自動更新機能付きの安定した QR コード接続。
- 📩 テキスト、画像、オーディオ (PTT/iPhone)、ビデオ、ドキュメントの送信。
- 🔘 Webhook コールバック付きのインタラクティブなボタン（ステートレス）。
- 💖 ボタンクリック時の自動絵文字リアクション。
- 🔐 API キーとリモートセッション管理による完全なセキュリティ。
