## 什么是 ZapUnlocked-API？

**ZapUnlocked-API** 是一个专业的、**100% 免费且开源**的解决方案，旨在将 WhatsApp 转换为强大的自动化工具。该 API 建立在 **Baileys** 引擎之上，提供简单的 REST 接口来管理会话、发送复杂媒体，并在无需繁重数据库的情况下创建智能交互。

> [!TIP]
> 非常适合寻求快速集成机器人、通知和自动客勤服务系统的开发人员。

---

## 🛠️ API 超强功能 🔥

- ✅ **弹性连接**：稳定的二维码，带有浏览器动态自动刷新系统。
- 📩 **完整消息传递**：发送文本、图片、视频、文档（最高 2GB）和联系人。
- 🎙️ **智能音频**：发送 PTT（语音消息），具有原生 iPhone 支持（自动转换为 OGG Opus）。
- 🔘 **无状态交互**：可定制按钮，按钮 ID 中集成了加密的 Webhook 回调。
- 💖 **自动回应**：设置表情符号，在用户点击按钮时立即做出回应。
- 🔐 **强大的安全性**：通过 API 密钥（标头或查询参数）进行身份验证，并具有会话清除的安全注销功能。
- 🔄 **自动恢复**：自动检测断开连接并重新启动身份验证流程的系统。
