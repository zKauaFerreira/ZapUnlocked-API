Atualize o nome de exibição e a foto de perfil do próprio Bot.

### Autenticação

<ParamField path="railway_domain" type="string" required placeholder="seu-projeto.up.railway.app">
  URL do seu projeto na Railway (sem https:// e sem a barra final).
</ParamField>

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

### Body

<ParamField body="name" type="string">
  Novo nome de exibição para o perfil do Bot.
</ParamField>

<ParamField body="newProfilePictureUrl" type="string">
  URL pública de uma imagem (JPEG/PNG) para definir como nova foto de perfil.
</ParamField>

<Note>
  Você deve enviar pelo menos um dos campos: `name` ou `newProfilePictureUrl`.
</Note>

### Response

<ResponseField name="success" type="boolean">
  Indica sucesso.
</ResponseField>

<ResponseField name="updated" type="array">
  Lista de campos que foram atualizados.
</ResponseField>
