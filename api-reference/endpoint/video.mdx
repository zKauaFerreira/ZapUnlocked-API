Envia um vídeo. Se for menor que 15MB, envia como vídeo normal (com compressão). Se for maior, envia como documento (alta qualidade).

### Corpo da Requisição (Body)

<ParamField path="railway_domain" type="string" required placeholder="seu-projeto.up.railway.app">
  URL do seu projeto na Railway (sem https:// e sem a barra final).
</ParamField>

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país.
</ParamField>

<ParamField body="video_url" type="string" required>
  URL pública direta do vídeo (mp4).
</ParamField>

<ParamField body="caption" type="string">
  Legenda opcional para o vídeo.
</ParamField>

<ParamField body="gifPlayback" type="boolean" default="false">
  Se `true`, envia o vídeo como um GIF (em loop e sem som).
</ParamField>

<ParamField body="ptv" type="boolean" default="false">
  Se `true`, envia como vídeo redondo (curto).
</ParamField>

<ParamField body="asDocument" type="boolean" default="false">
  Força o envio como documento (2GB limit).
</ParamField>



### Resposta

<ResponseExample>
```json 200 Sucesso
{
  "success": true,
  "message": "Vídeo enviado com sucesso ✅"
}
```

```json 401 Erro
{
  "error": "Unauthorized",
  "message": "API Key inválida ou ausente"
}
```
</ResponseExample>
