Atualize suas configurações de privacidade do WhatsApp, como quem pode ver seu visto por último, foto de perfil, etc.

### Autenticação

<ParamField path="railway_domain" type="string" required placeholder="seu-projeto.up.railway.app">
  URL do seu projeto na Railway (sem https:// e sem a barra final).
</ParamField>

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

### Body

<ParamField body="lastSeen" type="string">
  Quem pode ver seu "visto por último". Opções: `all`, `contacts`, `contact_blacklist`, `none`.
</ParamField>

<ParamField body="online" type="string">
  Quem pode ver quando você está online. Opções: `all`, `match_last_seen`.
</ParamField>

<ParamField body="profilePicture" type="string">
  Quem pode ver sua foto de perfil. Opções: `all`, `contacts`, `contact_blacklist`, `none`.
</ParamField>

<ParamField body="readReceipts" type="string">
  Quem pode ver confirmação de leitura (o check azul). Opções: `all`, `none`.
</ParamField>

<ParamField body="groups" type="string">
  Quem pode te adicionar em grupos. Opções: `all`, `contacts`, `contact_blacklist`, `none`.
</ParamField>

<ParamField body="about" type="string">
  Define o seu recado (status textual). Suporta placeholders: `{{day}}`, `{{mon}}`, `{{yea}}`, `{{hou}}`, `{{min}}`, `{{sec}}`.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
    "success": true,
    "updated": [
        "Visto por último: all",
        "Recado/About alterado para: \"Online desde: 15/01/2026 - 00:02\""
    ]
}
```

</ResponseExample>
