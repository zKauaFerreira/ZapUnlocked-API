| Recurso | Limite |

## <Icon icon="circle-check" iconType="sharp-regular" color="#9d4fe6" size={22} /> Conclusão

A **ZapUnlocked-API** consome aproximadamente **US$ 0.51 por mês** no plano gratuito da Railway, deixando uma margem confortável para picos de uso.

**Principais pontos**:
- **RAM** é o maior custo individual
- **CPU** e **Network** têm impacto **mínimo**
- **Armazenamento** permanece dentro do **limite** de 500 MB
- O **projeto** opera com **folga** no **crédito gratuito** de US$ 1.00

<Warning>
  Para manter os custos baixos, evite sincronizar históricos muito grandes e monitore o uso de armazenamento regularmente.
</Warning>
